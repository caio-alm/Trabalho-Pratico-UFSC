# Trabalhos Praticos (Linguagem de Progrmação I)


Este projeto foi desenvolvido para a disciplina de **Linguagem de Programação I**.

O objetivo foi desenvolver um sistema de gerenciamento de jogadores e de partidas de futebol.

**[Clique aqui para ler o Enunciado Completo do primeiro trabalho(PDF)](./Enunciado1.pdf)**

---

## Scripts Disponíveis

| Arquivo | Descrição |
| :--- | :--- |
| **`trabalhoPratico1.c`** | Realiza o cadastro de jogadores e partidas, visualização desses dados e relatórios. |
| **`trabalho_pratico2`** | Faz as mesmas coisas que o primeiro pôrem com o uso de listas encadeadas, persistencia de dados, opçoes de saída na tela e por arquivo CSV. O codigo desta vez tambem está modularizado em arquivos .h e .c e é compilado a partir de um Makefile.|

## Funcionalidades
- Cadastro de jogadores.
- Cadastro de partidas.
- Relatórios.
- Buscas.

## Tecnologias Utilizadas
- **Linguagem:** C.
